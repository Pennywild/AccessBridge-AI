# AccessBridge AI

AccessBridge AI is an AI-powered platform that improves digital accessibility by analyzing and transforming websites in real time.

## Vision
Bridge the gap between underserved communities and digital access using AI.

## Features (MVP)
- Website accessibility scanning
- Auto alt-text generation
- Content simplification
- Accessibility compliance insights

## Built For
- Underserved communities
- Small businesses
- Nonprofits
